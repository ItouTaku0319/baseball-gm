## 1. 打球の物理特性

### 1.1 打球速度（Exit Velocity）

| 指標 | 数値 |
|------|------|
| MLB平均 | 88-89 mph (141-143 km/h) |
| ハードヒット閾値 | 95 mph (153 km/h) |
| エリート打者のEV90 | 111-112 mph |
| 非力な打者のEV90 | 96-97 mph |

**打球速度とアウト率の関係:**
- 95mph以上: 打率 .506、長打率 1.008
- 95mph未満: 打率 .221、長打率 .261
- 75mphがゴロが内野を抜けるかどうかの分岐点

### 1.2 打球角度（Launch Angle）

| 打球タイプ | 角度範囲 | 発生割合 |
|-----------|---------|---------|
| ゴロ (GB) | 10度未満 | **43%** |
| ライナー (LD) | 10-25度 | **21%** |
| フライ (FB) | 25-50度 | **36%** |
| ポップフライ | 50度以上 | (FBに含む) |

スウィートスポット（最も安打になりやすい角度帯）: 8-32度
ホームランが多い角度: 20-25度

### 1.3 打球方向（Pull/Center/Oppo）

| 方向 | 全打球 | ゴロのみ | フライのみ |
|------|--------|---------|-----------|
| プル（引っ張り） | **41%** | **52%** | **23%** |
| センター | **35%** | - | - |
| オポジット | **24%** | - | - |

- ゴロはプル方向に偏る（52%）
- フライは比較的均等に散る
- 長打の53%はプル方向

### 1.4 打球種類別のアウト率

| 打球タイプ | BABIP | アウト率 |
|-----------|-------|---------|
| ゴロ | .236 | **76%** |
| フライ | .118 | **88%** |
| ライナー | .678 | **32%** |

## 4. 打球処理の行動パターン

### 4.1 内野ゴロの処理

**処理の流れ:**
1. 反応（0.2-0.4秒）→ 打球方向を判断
2. 移動（正面/バックハンド/フォアハンド）→ 捕球
3. トランスファー（グラブ→投げる手）→ 送球
4. 送球 → 一塁手（または他の塁）が捕球

**タイミング目標:** 打球インパクト→一塁到達まで **4.0秒以内**
- 右打者の一塁到達: 平均4.3秒
- 左打者の一塁到達: 平均4.2秒

**送球の物理:**
- バウンド送球 vs ノーバウンド: 時間差はわずか5-10ms
- 110ft以上 かつ 66mph以下 の場合のみバウンド送球が有利
- それ以外はノーバウンドが同等以上

### 4.2 ダブルプレー

**発生率:**
- 1試合あたり: 約0.8回/チーム
- 走者1塁・2アウト未満で打席の10-11%がGIDP
- 遅い打者は20%超、速い打者は5%程度

**主なパターン:**
| パターン | 流れ | 特徴 |
|---------|------|------|
| 6-4-3 | SS→2B→1B | 最も一般的。ボールが自然に流れる |
| 4-6-3 | 2B→SS→1B | SSのピボット（ターン）が見せ場 |
| 5-4-3 | 3B→2B→1B | 3Bの強い送球が必要 |
| 3-6-3 | 1B→SS→1B | 1Bが2塁に送球→SSがリレー |
| 1-6-3 | P→SS→1B | 投手がゴロ処理 |

**DP時の内野手の位置:** 通常位置から3-4歩前+二塁寄り

### 4.3 外野フライの処理

**追球の仕方:**
- ルーティン（Catch Prob 90%+）: ドリフト（微調整しながら歩み寄る）
- 難しい（Catch Prob 50%以下）: 全力疾走。ルートは二の次
- 背走: 最も難しい。背中を向けて全力疾走→最後に振り返る

**ルート効率:**
- エリート外野手: 95%以上（実際の走路 / 直線距離）
- 初動の速さ（Jump）がルート効率より守備力を決定づける

**背走ペナルティ:** 通常速度の85-90%

**ダイビングキャッチ:**
- 全打球の約10%でダイブが必要
- エリートの成功率: 約90%
- 5スター機会（Catch Prob 0-25%）の実際の捕球率: 7.1%
- 発生条件: 走っても間に合わないが飛べば届く距離（1-3ft足りない）

### 4.4 壁際のプレー

- ウォーニングトラック幅: 約16ft (5m)
- フェンスから16ft以内で減速開始
- フェンスから5ft以内でジャンプキャッチまたは諦め判断

## 5. 送球判断ロジック

### 5.1 基本ルール: 「2つ先の塁」

| 走者状況 | 送球先 |
|---------|--------|
| 走者なし | 一塁（打者走者） |
| 走者1塁 | 二塁（併殺狙い）→ 一塁 |
| 走者2塁 | 本塁（得点阻止） |
| 走者1,2塁 | 二塁（併殺）or 本塁（得点阻止） |
| 満塁 | 本塁（フォースプレー）→ 一塁 |

### 5.2 修正要因

| 要因 | 影響 |
|------|------|
| **0アウト** | 併殺優先（二塁→一塁） |
| **1アウト** | 状況次第（リード走者 or 併殺） |
| **2アウト** | 最も確実なアウト（通常一塁）を優先 |
| **大差リード** | 確実なアウト優先（リスク回避） |
| **僅差/同点** | 得点阻止を最優先 |
| **終盤** | 得点阻止の重要度がさらに上がる |
| **走者の走力** | 足の速い走者には無理な送球をしない |
| **打球の深さ** | 深いゴロは一塁のみ、浅いゴロはリード走者を狙える |

### 5.3 外野手の送球判断

- 基本: ランナーの「2つ先の塁」方向にカットマンへ送球
- 走者を進塁させないことを優先（得点圏に進めない）
- 犠飛時は捕球体勢でモメンタムを送球方向に残す

## 6. 中継（カットオフ）プレー

### 6.1 カットマンの担当

| 打球方向 | 中継担当 | もう片方 |
|---------|---------|---------|
| LF方向 (0-45度) | **SS** | 2B→二塁カバー |
| CF方向 (30-60度) | **SS** | 2B→二塁方向 |
| RF方向 (45-90度) | **2B** | SS→二塁カバー |

**重要:** 両方が中継に行くことはない。片方は必ず二塁カバー。

### 6.2 中継位置の目標

| 走者状況 | 中継位置 |
|---------|---------|
| 走者なし | 二塁送球ラインの40%地点 |
| 走者1塁 | 三塁送球ライン |
| 走者2塁以上 | **ホーム送球ライン**の40%地点 |
| 走者3塁(犠飛) | **ホーム送球ライン** |

### 6.3 中継の物理

- 同じ腕の強さなら距離を等分するのが最適
- 腕の強さが違う場合は、強い腕が長い距離を投げる
- 空気抵抗により、2回に分けた方がトータル時間が短い
- 中継の精度目標: 約3ft × 3ft（胸〜頭の高さ）

### 6.4 カットするかスルーするかの判断

| コール | 意味 |
|--------|------|
| (塁番号のみ) | スルー（送球をそのまま通す） |
| 「カット!」+塁番号 | カットして指定の塁に送球 |
| 「カット!」のみ | カットしてボール保持 |

判断基準:
- 送球が逸れている → カット
- リード走者がアウトにできない → カットして他の走者を狙う
- 送球がベースまで届かない → カット

## 7. ポジション別の特殊行動

### 7.1 投手 (P)

| 状況 | 行動 |
|------|------|
| 右方向のゴロ（1Bが離れた） | **一塁カバー（バナナルート）** |
| バント | 前進して処理（マウンド付近のバント） |
| 外野フライ/ヒット | ホーム方向に下がる（バックアップ） |
| 走者2塁以上で外野ヒット | 三塁線〜本塁中間に移動→展開を見て判断 |
| 自分の前のゴロ | 捕球→一塁送球 |

**バナナルート:** 一塁に直線で走るのではなく、ファウルライン上の約2/3地点を目指し、ライン沿いに並行して走って一塁ベース内側を踏む。直線だと走者と衝突+送球が背中側になる。

### 7.2 捕手 (C)

| 状況 | 行動 |
|------|------|
| 基本 | ホーム待機 |
| 走者2塁以上 | ホームでタッチプレー準備 |
| 走者なし+内野ゴロ | 一塁後方へバックアップ |
| バント | 全体の司令塔。到達順で判断、迷ったら一塁送球指示 |
| 盗塁 | 二塁送球（ポップタイム参照） |

**ポップタイム（二塁盗塁阻止）:**
| 指標 | 数値 |
|------|------|
| MLB平均 | **2.01秒** |
| エリート | 1.89秒以下 |
| 劣る | 2.14秒以上 |
| 内訳: エクスチェンジ | 0.73秒 (36.6%) |
| 内訳: ボール飛行 | 約1.28秒 (63.4%) |

### 7.3 一塁手 (1B)

| 状況 | 行動 |
|------|------|
| 基本 | 一塁ベースカバー |
| 自分の方向のゴロ | 捕球→投手が一塁カバーに来るので、トスまたは自分でベースへ |
| RF線方向の打球 | ボール方向に移動することがある |
| バント守備 | 前進チャージ |

### 7.4 二塁手 (2B) / 遊撃手 (SS)

| 状況 | 2Bの動き | SSの動き |
|------|---------|---------|
| LF方向外野ヒット | 二塁カバー | **中継（カットマン）** |
| RF方向外野ヒット | **中継（カットマン）** | 二塁カバー |
| 内野ゴロ（相手側） | 二塁カバー | 二塁カバー |
| DP時 | ベース寄り+前 | ベース寄り+前 |

### 7.5 三塁手 (3B)

| 状況 | 行動 |
|------|------|
| ほぼ常に | 三塁ベースカバー |
| 走者2塁のLF線深め長打 | 三塁-ホーム間の第二中継に入ることがある |
| バント守備 | 前進チャージ |

### 7.6 外野手共通

| 状況 | 行動 |
|------|------|
| 自分の方向のフライ | **捕球** |
| 隣接OFの方向のフライ | バックアップ位置まで移動 |
| 反対側OFの方向のフライ | 軽くドリフト（少し寄る程度） |

**フライの優先ルール:**
1. CF（中堅手）が最高優先
2. コーナーOF（LF/RF）が内野手より優先
3. 前進する選手が優先（前に来る方がキャッチしやすい）
4. 内野手のポップアップ優先: SS > 2B > 3B > 1B > P

## 8. エラーモデル

### 8.1 基本エラー率

| 打球タイプ | 基本エラー率 | 備考 |
|-----------|-----------|------|
| ゴロ | **2-3%** | インプレー打球全体の平均 |
| フライ | **0.3-0.6%** | ゴロの約1/5-1/10 |

### 8.2 状況別エラー率（ゴロ）

| 状況 | エラー率 |
|------|---------|
| 走者なし、0アウト | 1.4% |
| 走者あり、0アウト | 2.1% |
| 一塁走者、2アウト | 3.41%（最低） |
| 二三塁走者、1アウト | **9.54%**（最高） |

### 8.3 ポジション別の傾向

- **SS・3B**: 最もエラーが多い（難しいゴロ+長距離送球）
- **1B**: 捕球機会多いが比較的少ない
- **OF**: 稀（主に送球エラー）
- **P**: 守備機会自体が限られる

### 8.4 エラーの要因

- 走者がいる場面でプレッシャー増→エラー率上昇
- 長距離送球（SS→1B、3B→1B）で送球エラー多い
- 難しい体勢（逆シングル、背走キャッチ）で上昇
- ゴロのエラー率はフライの5-15倍

### 8.5 リーグ全体

- 2023年MLB: 1試合あたり **0.52エラー**（過去最低）
- インプレー打球に対するエラー率: 約2.25%

## 9. タッチアップ（犠飛）

### 9.1 タイミングの計算

| 要素 | 時間 |
|------|------|
| フライ滞空時間 | 4.0-5.5秒 |
| 捕球→リリース | **0.8-1.5秒**（クロウホップ含む） |
| 送球飛行（300ft, 85mph） | **約2.4秒** |
| ランナー 3塁→本塁 | **約3.5-4.0秒** |

**判定:** 「捕球→リリース + 送球飛行時間」 vs 「ランナーの走塁時間」

### 9.2 外野手の捕球体勢

- モメンタムを送球方向に残して捕球（投げる側の足を後ろに）
- 捕球後1-2歩のクロウホップで助走
- フルスローで中継マンまたは直接ベースへ

---

## 10. ランダウンプレー（挟殺）

### 10.1 基本の動き

1. ボール保持者が走者に向かって全力で走る
2. 走者がコミットしたら即座にパートナーに送球
3. パートナーはベース付近でスタンバイ→送球を受けてタッグ

### 10.2 ルール

- **最小限の送球（理想は2回以下）** で完了
- 2人の野手は走者の同じ側に位置（送球が走者に当たるのを防ぐ）
- ベースカバー役はベース手前で待機
- 走者を**元の塁に追い返す**のが基本

## 11. 内野安打

### 11.1 発生条件

- 打者の走力（Sprint Speed）が最大の要因
- 左打者は一塁までの距離が短いため内野安打率が高い
- 打球速度75mph未満のゴロは内野手の処理時間に余裕がある

### 11.2 走力の影響

| 走力 | 打率への影響 |
|------|-----------|
| MLB平均 (27 ft/s) | 基準 |
| 平均以上（各ティア） | +5〜+16ポイント |
| 平均以下（各ティア） | -5〜-16ポイント |
