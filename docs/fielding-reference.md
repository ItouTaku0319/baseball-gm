## 1. カバーリング・フォーメーション

出典元: `fielding-coverage.md`
出典: 工藤康博「守備陣の動き・カバーリング」シリーズ (note.com/baseballtheory)

### 基本原則

- **全員が必ず動く。何もしないポジションは存在しない**
- 打球を処理しない野手は「中継」「ベースカバー」「バックアップ」のいずれかを行う
- 遠方の外野手は**少し寄る程度**（全力疾走はしない）

### 走者1塁 — 外野ヒット/二塁打

走者1塁の場合も基本は走者なしと同構造。中継はホーム送球ラインではなく三塁方向。

### 走者2塁以上 — 外野ヒット（ホーム送球あり）

走者2塁の場合、中継は**ホームへの送球ライン**に入る。
走者3塁（犠飛パターン）も同様にホーム送球ラインの中継。

### パターンまとめ（実装用）

#### 中継（カットマン）のルール

| 打球方向 | 中継担当 | もう片方 |
|---|---|---|
| LF方向（direction 0-45°） | **SS(6)** | 2B(4)→二塁カバー |
| CF方向（direction 30-60°） | **SS(6)** | 2B(4)→二塁方向 |
| RF方向（direction 45-90°） | **2B(4)** | SS(6)→二塁カバー |

**重要**: 中継に入らない方は必ず二塁ベースカバーに行く。両方が中継に行くことはない。

#### 走者状況による中継位置の違い

| 走者状況 | 中継位置の目標 |
|---|---|
| 走者なし | 二塁送球ライン（打球→二塁の40%地点） |
| 走者1塁 | 三塁送球ライン |
| 走者2塁以上 | **ホーム送球ライン**（打球→ホームの40%地点） |

#### 投手(P)の動き

| 状況 | 動き |
|---|---|
| 外野フライ/ヒット | ホーム方向に下がる（バックアップ） |
| 1Bゴロ | 一塁ベースカバー（弧を描いて走る） |
| その他のゴロ | マウンド付近→ホーム方向バックアップ |

## 2. 守備AIシミュレーション実データ・知見

出典元: `fielding-ai-reference.md`

### 投手のフィールディング (PFP)

- 投手のMLBプットアウト割合: 約**2.0%**
- PFP訓練内容: バント処理、ゴロ処理、一塁カバー、バックアップ、コミュニケーション

### 投手の一塁カバー（バナナルート）

- 一塁手が右方向ゴロで離れた時、バント処理で前に出た時
- ファウルライン上の約2/3地点をまず目指し、ライン沿いに並行して走り、一塁ベース内側を右足で踏む

### 投手のバックアップ

- バックアップ位置はベースの後方深く、送球の延長線上
- 三塁線と本塁の中間（ファウル地域）に移動→展開を見て判断

### 捕手の送球（ポップタイム）

| 指標 | 数値 |
|---|---|
| MLB平均ポップタイム | **2.01秒** |
| エリート | 1.89秒以下 |
| エクスチェンジ（捕球→リリース） | 0.73秒 |
| ボール飛行時間 | 約1.28秒 |

### 送球先の判断ロジック

基本は「2つ先の塁に投げる」ルール + アウトカウント・得点差で修正。

### エラーモデル

- MLB平均: インプレー打球に対するエラー率約2.25%
- ゴロのエラー率はフライの約5〜15倍
- SS・3Bが最もエラーが多い

### Statcast打球データ

- **打球速度**: MLB平均約88-89mph、ハードヒット閾値95mph+
- **打球角度**: ゴロ<10°、ライナー10-25°、フライ25-50°、ポップ50°+
- **方向分布**: プル41%、センター35%、オポ24%
- **打球タイプ**: ゴロ43%、フライ36%、ライナー21%

### 打球タイプ別アウト率

| 打球タイプ | BABIP | アウト率 |
|---|---|---|
| ゴロ | .236 | 約76.4% |
| フライ | .118 | 約88.2% |
| ライナー | .678 | 約32.2% |

## 3. 守備シチュエーション完全ガイド

出典元: `defensive-situations-complete.md`
出典: "Defensive Situations and Strategies" PDF + BaseballPositive Coaching Guide

### 3つの個人責任

1. **Ball**: 打球を処理する
2. **Base**: ベースをカバーする
3. **Back-up**: 送球のバックアップに回る

### カットオフマンの法則

| 打球方向 | ランナーなし | ランナー1塁/1,3塁 | ランナー2塁/1,2塁/満塁 |
|---|---|---|---|
| レフト | SS→二塁 | SS→三塁 | **3B→ホーム** |
| センター | SS→二塁 | SS→三塁 | **1B→ホーム** + SS→三塁 |
| ライト | 2B→二塁 | SS→三塁 | **1B→ホーム** + SS→三塁 |

### 長打時のカットオフ

| 打球方向 | ランナーなし | ランナーあり |
|---|---|---|
| 左翼線/左中間 | SS→三塁 | SS→ホーム |
| 右中間/右翼線 | 2B→三塁 | 2B→ホーム |

### 一般法則

- **左方向の打球**: SS がカットオフ
- **右方向の打球**: 2B がカットオフ
- **ランナー2塁以降**: 送球先がホーム → カットオフマンが変わる（3B or 1B）
- **長打**: SS/2Bがリレー。もう一方がトレイル（後方待機で指示出し）

### バント守備

捕手が指示者。P・1B・3Bがチャージ、2Bが一塁カバー、SSが二塁カバー。

### ポップフライ優先ルール

**外野手 > 内野手 > P > C**

## 4. 守備フォーメーション参考資料

出典元: `fielding-formation-reference.md`

### 内野守備ポジショニング深度

| 深度 | 使用場面 | 位置 |
|---|---|---|
| Position 1: 標準 | ランナーなし/2アウト | 外野芝5-15ft手前 |
| Position 2: DP | 2アウト未満+ランナー1塁 | 標準より3歩前+二塁寄り3歩 |
| Position 3: ハーフウェイ | ランナー1,3塁 | ベースライン上 |
| Position 4: 前進 | ランナー3塁+得点阻止 | 内野芝とダートの境目 |

### ゴロ時のDP体制（ランナー1塁）

**二遊間のピボット判定**: 打球方向でSS/2Bどちらが二塁ベースに入るか決まる
- 三塁方向(左)のゴロ → **2Bが二塁ベース入り**（ピボットマン）
- 一塁方向(右)のゴロ → **SSが二塁ベース入り**（ピボットマン）

### DPの詳細データ（Statcast実データ 2024年4月）

- **ランナー1塁+2アウト未満のゴロ**: DP率 **31.6%**
- SS(36.2%) > 2B(26.4%) > 3B(20.6%)がDP処理の中心
- 速い打球ほどDP率が高い（90mph以上で37%）

### 野手の標準位置

| ポジション | ホームからの距離 | 角度 |
|---|---|---|
| LF | 270-280ft | -33°〜-21° |
| CF | 300-320ft | -10°〜+10° |
| RF | 270-280ft | +21°〜+33° |

## 5. 外野守備AI実データ・知見

出典元: `outfield-defense-research.md`

### Statcast Jumpメトリクス

| フェーズ | 時間窓 | 内容 |
|---|---|---|
| Reaction | 最初の1.5秒 | 打球への反応・最初の一歩 |
| Burst | 1.5〜3.0秒 | 加速フェーズ |
| Route | 全3.0秒 | 移動距離 vs 正しい方向の比率 |

**重要**: リアクション(初動)が最も重要。ルート効率よりも初動の速さが守備力を決定づける。

|---|---|---|
| エリート | 30.0+ | 9.14+ |
| リーグ平均 | 27.0 | 8.23 |
| 遅い | 23.0-25.0 | 7.0-7.6 |

### 外野からの送球（Arm Strength）

| カテゴリ | 速度 (mph) |
|---|---|
| エリート | 95-100+ |
| リーグ平均 | 約85 |
| 弱い腕 | 75-80 |

### フライボール優先ルール

1. **CF** > 2. コーナーOF > 3. 内野手

### Catch Probability

| スター | Catch Probability | 難易度 |
|---|---|---|
| 5 | 0-25% | 超高難度 |
| 4 | 26-50% | 高難度 |
| 3 | 51-75% | 中難度 |
| 2 | 76-90% | やや難 |
| 1 | 91-95% | ルーティンに近い |

### シミュレーション用の表現

| 人間の行動 | シミュレーションでの表現 |
|---|---|
| 反応遅延 (0.5秒) | 動き出しに遅延を入れる |
| 不完全なルート | ルート効率パラメータ(95-98%) |
| 全力疾走 vs ドリフト | Catch Probabilityに応じて速度切替 |
| 背走の難しさ | 背走時に速度ペナルティ(85-90%) |
| ダイビング | 距離1-3ft不足時にダイブ判定 |
| 壁際の減速 | フェンスから16ft以内で最高速度制限 |
